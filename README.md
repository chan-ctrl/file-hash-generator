# file-hash-generator
A simple SHA-256 hash generator built using python

## What this project does
This program generates a SHA-256 hash for user input.

## Features
- Takes user input
- Genrates SHA-256 hash
- Displays the hash value

## Technologies used
- Python 3
- hashlib library

## How to Run
python hash_generator.py
